# **Практична робота №9**  
**Тема:** Функція Auto layout у Figma. Створення картки товару (послуги)

## **Мета роботи**
Ознайомитися з можливостями **Auto layout** у Figma та навчитися створювати зручні, адаптивні картки товарів або послуг із використанням цього інструмента.  
Закріпити навички роботи з **іконками**, **зображеннями**, **текстом** та **компонентами**.

## **Завдання**
Створити малу **картку товару**, використовуючи функцію **Auto layout** у Figma.  
Під час оформлення застосувати відповідне зображення, текст, іконки та елементи керування (кнопки).

## **Плагіни, що використовувалися**
- **Unsplash** — імпорт зображень різних категорій;  
- **Feather Icons / Iconsax 6000+** — імпорт іконок;  
- **Remove BG** — автоматичне видалення фону із зображень.

## **Хід роботи**
1. Опрацював розділ із “Керівництва користувача Figma” про Auto layout.  
2. Створив основу картки з прямокутників і текстових блоків.  
3. Додав Auto layout для зручного вирівнювання елементів.  
4. Вставив іконки “серце”, “кошик” і кнопки дій.  
5. Використав кольорову палітру в теплих тонах і шрифт без зарубок.  
6. Вирівняв всі елементи для гармонійного візуального вигляду.
